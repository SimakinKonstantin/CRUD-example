# Web-сервис сотрудников, сделанный на Golang

## Функционал
1. Возможность добавления сотрудников, в ответ должен приходить Id добавленного сотрудника;
2. Возможность удаления сотрудников по Id;
3. Возможность выводить список сотрудников для указанной компании;
4. Возможность все список сотрудников для указанного отдела компании;
5. Возможность изменения определенных полей сотрудников по его Id.

Модель сотрудника:
   ```
   {
       Id int
       Name string
       Surname string
       Phone string
       CompanyId int
       Passport {
           Type string
           Number string
       }
       Department {
           Name string
           Phone string
       }
   }
   ```
   
### Инструкция по запуску:
1. Определить переменную окружения ```POSTGRES_CONNECT_URL``` со ссылкой на подключение к БД.
2. В ```./internal/config/config.go``` указать значение ServerAddr - адреса сервера.
3. Запустить миграцию, создающую таблицу через ```cd ./migrations && goose up``` либо
вручную выполнив sql код до ```-- +goose Down```.
4. Запустить ```./cmd/main.go```.
